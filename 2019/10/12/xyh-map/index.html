鑫优会《创业流程及知识图》

发表于 2019-10-12 更新于 2019-10-17

三大流程，百个知识点，指明清晰的事业路径！
每个人都可以成功创业！

市场业务流程

一、引流

现有流量

  • 现有群资源
    项目群学习群生活群
  • 微信好友
  • 身边人脉

新流量

  • 微信群
    直销群微商群社交电商群
    付费课程群其它项目群
  • 文章
    百家号头条号搜狐知乎简书
    爱喇叭云同盟豆瓣赶集58
  • 其它
    小红书抖音微信公众号
    今日头条百度贴吧百度知道

二、拉群

重点

  1. 每天坚持行动
  2. 保证客源精准
  3. 多号交叉行动
  4. 保持合理频率

拉群工具

  • WeTool免费版 → 下载链接
  • WeTool企业版 → 下载链接

拉群文案

  • 拉群前必须给每个人发文案、文章
  • 没有文案，别人无法了解群，也就不关注群，进群就乱发广告

进群欢迎语

  • 提醒群分享内容，吸引客户关注群并及时听课
  • 提醒群规
  • 注意频率（设置可调）

三、转化（公开课）

1. 新人进群提醒

2. 确定全天群内分发内容

  • 文章
  • 文案
  • 语音链接

3. 新群老群区别对待

  • 新群
    进群提醒，主讲简介，课程相关文章
  • 老群
    早、中、晚分别发不同内容（事先确定好）

4. 开课前造势

  • 中午开始发通知（1、2小时发一次）
  • 新人进群提醒
  • 课前1小时大力造势：主讲简介、文章、导图、文案、红包（每天5个红包，间歇安排）

5. 课中互动

  • 简洁、真实、适量

6. 课后互动

  • 学习感悟
  • 正式课截图
  • 付款截图
    内容诱惑，场面火爆

四、成交（付费课）

1. 分享资料

  • 拉群文案
  • 海报
  • 文章

2. 群公告

  • 课程主题
  • 群规

3. 开课前氛围

  • 新人进群自我介绍（文字）
  • 主讲人简介
  • 课程宣导文章
  • 红包

4. 听课进度跟进

  • 学员一对一通知

5. 课程感悟分享

  • 学习后引导学员发学习感悟
  • 正式会员带头，给予适当奖励

五、裂变（正式会员）

1. 新人启动（五步曲）

  • 推荐人，辅导老师第一次交流（了解新人）
  • 了解鑫优会（初步学习资料）
  • 挖掘需求，确立梦想（过去、现在、未来）
  • 建立信念，看到希望（为什么在鑫优会可以）
  • 规划方案，确立目标（从0开始怎样走向成功）

2. 确立第一步目标

  • 五星，直推5个会员

3. 明确第一步行动

  • 引流+拉群

4. 学习“建立万人团队计划”

5. 按照“建立万人团队计划”，一步一步上五星

6. 小组会

心态
行动
认可
挫折预防
文化宣导
感恩付出责任团队精神

7. 万人团队

  • 培养核心
  • 成立小组
  • 学习团队管理
  • 裂变万人团队

六、五星达成

1. 建立小组

  • 小组群
    群规群管理
  • 小组会
    每周1-2次
  • 小组会主题
    模式运作心态目标希望挫折预防
  • 文化建设
    感恩合作团结付出
  • 常见问题解答

2. 培养核心

找到5-10个核心，统一思想，统一目标，统一行动
共同价值观
认可积极感恩合作担当企图心

3. 裂变

  • 统一思想（重点是核心）
  • 复制行动
  • 帮助伙伴上五星
  • 凝聚团队（文化+价值观）

七、万人团队

1. 找到5-10个核心

认同心态决心行动感恩担当企图心

2. 培养核心上5星

  • 培训
  • 核心会议
  • 指导行动
  • 协助支持

3. 深化文化建设

认同价值观感恩担当合作

  • 培训
  • 核心圈
  • 命运共同体

公司发展流程

千人领袖 | 百人创业孵化基地

  1. 市场根基，事业稳健的基础（重点培养+重点股权激励）
  2. 以平台为终生事业，成就一生梦想
  3. 企业成功，大家成功，共同创业，事业共同体
  4. 实现一生财富自由
  5. 成为真正的成功者（财富+个人能力影响力）

万人领导 | 千人创业孵化基地

  1. 市场中坚力量，做大做强的基础
  2. 以平台为事业，赚到财富，实现小康，实现生活自由
  3. 提供向上发展的空间，部分积极向上的人成就一生事业
  4. 提供完善培训，让积极的人都能成为优秀的人

十万五星 | 万人股权

  1. 一线市场领导
  2. 轻松月入过万
  3. 自由工作，摆脱朝九晚五
  4. 万人持股，分享公司红利
  5. 提供向上发展的空间，让部分积极上进的人成就一生事业

百万分享者

  1. 利用零散时间也能赚钱
  2. 自己消费省钱，分享赚取零花钱
  3. 在团队找到另一个生活圈，另一个生活方式，让生活更丰富、更精彩、更有意义

千万会员

消费更省钱更放心
